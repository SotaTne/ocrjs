    Interface IImage

    Image representation independent of underlying implementation. Can be backed by cv.Mat (OpenCV), Canvas, or other formats.

    This interface follows the same principles as ITensor - it's a handle to image data that may reside in different backends (CPU, GPU, WebGL).
