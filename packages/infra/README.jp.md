# infra

## 概要

`infra` は、ocrjs においてOCR 実行に必要なインフラストラクチャを提供する **実装層**です。
この層は、`infra-contract` が定義する契約に基づき、特定の実行基盤（onnx / wasm / web worker / native runtime など）に依存した具体的なインフラストラクチャを実装します。
この層は、ドメインや実行ロジックが特定の実行基盤に依存しないようにするために存在します。

## 役割と位置づけ

`infra`は、絶対にocrjsのビジネスロジックを含まず、`infra-contract`が定義する契約に忠実に従う実装のみを提供します。
